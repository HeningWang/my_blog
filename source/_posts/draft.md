# Install Hexo
Prerequisites for installing hexo are: Git und Nodejs
To install them on Windows: [Git](https://git-scm.com/download/win), [Node.js](https://nodejs.org/en/)
using Windows OS:

```
npm install hexo
```

# Initialise the blog and some relevant init configurations
```

```
